高黎贡手工造纸博物馆 / Gao Li Gong Paper Museum

云南 / Yunnan_2013

www.papermuseum.cn

项目时间：2013.01

位于云南腾冲的高黎贡手工造纸博物馆，作为优秀的建筑设计项目曾获得众多奖项。网站的设计根据馆方的设计要求力求界面简单、功能清晰、突出手工博物馆本身的图片及环境结构，清新简洁，自然质朴

Start time: 2003.01

Gao Li Gong paper museum is located in Teng Chong, Yunnan province. The architecture wins lots of awards. We did the website design part. Simple clear page, functional, maximum showing the images of the museum itself, environment and natural are the main request from the client. After several discussions with client, we make it as it is.

摄影师陈漫个人网站 / Chen Man Personal Website

北京 / Beijing_2008

www.chenmaner.com

项目业主：陈漫
项目时间：2008

作为目前最炙手可热的时尚摄影师及视觉艺术家，陈漫已经是众多一线明星的御用设计师。此网站设计于2008年，目前网站已经改版，但仍然沿用当时动态影棚的概念，简约的风格，个性化的体验感受曾经深受客户青睐，并持续使用了数年时间。

Client: Chen Man
Start time: 2008

Chen Man is one of the most famous photographer and vision artist in China. Nowadays, she became as the private photographer and vision advisor for many Chinese super stars.
This website was finished in 2008 which is already upgrade now. But it still keeps the concept of the movie making room. Simple, clear, recognizable makes this website accepted by client and use it for several years.
